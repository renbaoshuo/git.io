# 自定义短链接生成

## 介绍

请移步 <https://baoshuo.ren/post/oKnaKLcDb/> 查看。

## 使用

本程序提供两种版本，网页版和命令行版。

### 网页版

需和 PHP 版 API 配合使用。

### 命令行版

`git-io-cli.py` 为命令行版源文件。

#### 运行参数

| 参数序号 | 是否必须 | 注释      |
| ------- | ------- | --------- |
| 1       | 否      | 长链接    |
| 2       | 否      | 自定义短链 |

若无参数，则进入交互模式。
